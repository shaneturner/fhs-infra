Forrest Hill Infrastructure - CLI Instructions
===============================================


RUNNING COMMANDS FROM THE HOST
-------------------------------
Run Craft, Composer, or Node commands without entering the container:

  Craft CLI
    docker exec fhs-craft-app php craft <command>

  Composer
    docker exec fhs-craft-app composer <command>

  pnpm
    docker exec fhs-craft-app pnpm <command>


ENTERING THE CONTAINER (Interactive Shell)
------------------------------------------
  docker exec -it fhs-craft-app sh


SUBMODULE SYNCHRONIZATION
--------------------------
Pull the latest changes from all subprojects and update the root repository pointer:

  ./scripts/sync-submodules.sh


PRODUCTION OPERATIONS
----------------------
Deploy / Update Production
  1. Pull the latest code:    git pull
  2. Rebuild and restart:     docker compose -f docker-compose.yml -f docker-compose.prod.yml up -d --build

Reload Caddy Configuration (apply Caddyfile changes without a full restart)
  docker exec fhs-caddy caddy reload --config /etc/caddy/Caddyfile

Performance & Cache Status
  curl -s -I https://forresthill.school.nz | grep -i 'cache-status'
